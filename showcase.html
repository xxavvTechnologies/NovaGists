What's New in Nova Gists

Discover the latest improvements and features we've added to make your note-taking experience even better.

Enhanced Profile Menu

We've redesigned the profile experience with a new dropdown menu that gives you quick access to your account settings and sign out option. The menu shows your profile picture, name, and email for easy account identification.

Improved Search Experience

The search functionality now includes a dedicated button in the header with a keyboard shortcut (Ctrl+K), making it easier to find your gists. The search modal supports both personal and community gist searches.

Mobile Optimizations

We've enhanced the mobile experience with better touch targets, improved keyboard handling, and a mobile-friendly profile menu that slides up from the bottom. The app now fully supports iOS safe areas and dynamic islands.

Refined User Interface

The entire application has received a visual refresh with smoother animations, better contrast, and more consistent spacing. Dark mode support has been enhanced for better visibility.

Security Improvements

Enhanced Firestore security rules now provide better protection for your data while maintaining the flexibility to share and collaborate when needed.

New Notification System

A completely redesigned notification system provides clearer feedback for your actions with support for success, error, warning, and loading states.

Technical Improvements

  • • Enhanced Progressive Web App (PWA) support
  • • Improved offline functionality
  • • Better performance with optimized animations
  • • Reduced bundle size for faster loading
  • • Improved accessibility throughout the app
